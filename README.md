# Hello World · 优雅呈现

一个纯 HTML/CSS 的“Hello World”展示页面，采用现代玻璃拟态设计风格，视觉效果简洁优雅。无需任何外部资源，打开即用。

## ✨ 效果预览

深色渐变背景上，居中显示一个带有磨砂玻璃质感的卡片，卡片内呈现渐变色“Hello World”文字。页面包含柔和光晕、装饰性圆点和微妙的悬停动画。

## 🚀 快速开始

### 直接运行
1. 将 `index.html` 文件下载到本地
2. **双击文件**，即可在默认浏览器中查看效果

### 使用 Live Server（推荐）
1. 用 VS Code 打开项目文件夹
2. 安装 **Live Server** 插件
3. 右键 `index.html` → 选择 **Open with Live Server**
